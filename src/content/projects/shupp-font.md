Shupp is a custom display font created as part of establishing a unique brand identity for shupp.dev. The font is used for main headers and the site logo, giving the portfolio a distinctive and personal touch.

## Features

- Custom designed glyphs
- Optimized for display use (headings, logos)
- OpenType format for broad compatibility
- Open source and freely available

## Usage

The font is currently used on shupp.dev for:

- **H1 and H2 headings**: Main page titles and section headers
- **Site logo**: The "shupp.dev" wordmark in the navigation

## Technical Details

- **Format**: OpenType (.otf)
- **License**: Open source
- **Repository**: Available on GitHub under the Thoughtful App Co. organization

## Installation

The font can be loaded via CSS `@font-face`:

```css
@font-face {
  font-family: 'Shupp';
  src: url('/fonts/Shupp.otf') format('opentype');
  font-weight: normal;
  font-style: normal;
  font-display: swap;
}
```
